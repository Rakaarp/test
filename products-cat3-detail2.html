Do you need workstation-class high performance graphics and an improved end-user experience for your virtual desktops? Would you like to gain the security and manageability benefits of a data center architecture? The HP ProLiant WS460c Gen9 Graphics Server Blade allows you to centralize your organization’s high-end desktop PCs and workstations for more business efficient end-user computing infrastructure while delivering the same superior user experience. Assisted with workstation-class high-performance graphics hardware, the server blade can be a single-user dedicated graphics server or host multiple users sharing resources through graphics virtualization.

And with NVIDIA® GRID virtualization technology, right sizing the GPU performance to its user needs is a breeze through simple software configuration. The HP ProLiant WS460c Gen9 Server Blade is your business efficient and high-performance end-user computing platform delivering superior user experience from the data center.

Operational Efficiency

  • The HP ProLiant WS460c Gen9 Graphics Server Blade has a high host density of up to 16 (or 8 expansion) blades and high graphics density of up to 48 GPUs or 512 users per 10U rack space.
  • Simplified and streamlined through converged architecture.
  • HP BladeSystem inherent power, cooling and cost efficiency.

GPU Accelerated End-User Computing

  • The HP ProLiant WS460c Gen9 Graphics Server Blade delivers a virtualized, full-fidelity desktop experience including support for 3D graphics applications.
  • Provides hardware accelerated graphics for workstation class performance.
  • Up to 70% performance increases with Intel® Xeon® E5-2600 v3 processors.
  • HP DDR4 SmartMemory at 2133 MHz (up to 1 TB): +14% - 33% throughput compared to competition and 35% lower power consumption than 1.5v DDR3 at the same bin speed.
  • Optional HP Smart Array P244br Controller with 1 GB Flash-Backed Write Cache (FBWC) DDR3 at 1866 MHz improves storage performance for demanding workloads.

Improved Resource Use and Access Flexibility

  • The HP WS460c Gen9 Graphics Server Blade provides professional-grade performance and industry leading GPU density with NVIDIA Tesla M6, Grid virtualization technology, and the HP MultiGPU carrier.
  • Removes graphics performance bottlenecks with graphics accelerated virtualized desktop environment.
  • Provides a high level of flexibility to meet changing performance needs.

Transforming Business Economics by Accelerating Service Delivery

  • HP OneView provides the Power of One—a single comprehensive view of the data center, managing hardware, software, firmware, and drivers.
  • HP Integrated Lights-Out (iLO) software features server lifecycle management advancements including iLO Federation, which remotely manages groups of servers at scale.

What's New!

  • New E5-4600 v3 Intel® Xeon® processors, HP DDR4 SmartMemory, and 12 Gb SAS.
  • Flexible storage options for an ideal balance of performance and price.
  • Two additional SFF drives, internal USB 3.0, optional dual micro-SD, and optional M.2 support deliver a variety of system boot options.
